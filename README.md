# City

Thiết kế dãy phố — ứng dụng **OpenGL 3.3** (C++17) xem cảnh `procedural_city_7.glb`, có menu **Dear ImGui** (tiếng Việt), thời gian trong ngày, sương mù và mưa (hậu kỳ).

Mã nguồn và CMake nằm trong thư mục [`project/`](project/). Hướng dẫn build: [`project/BUILD.md`](project/BUILD.md).

**Model 3D:** file `procedural_city_7.glb` **không** có trong repo (tránh giới hạn dung lượng). Sau khi clone, tự đặt file vào `project/assets/models/procedural_city_7.glb` (xem [`project/BUILD.md`](project/BUILD.md)).

Repository: [github.com/imxyanua/City](https://github.com/imxyanua/City)
